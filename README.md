## Clarifying some of the folders in src/js:

- **AIMover** – Contains code for moving different entities  
- **Missions** – Contains missions and quests (experimental)  
- **Effects** – Applies instant or over-time modifications to entity stats  
- **Engine** – Contains the game engine  
- **Items** – Objects that can be picked up  
- **Map** – Tools to create maps  
- **Objects** – World objects that cannot be picked up  
- **Pickups** – Items in picked-up state  
- **StatData** – Base values for entities and weapons, with modifiers
