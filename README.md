Clearfiying some of the folders:
AIMover - Contains code for  moving different entities
Missions - Contains missions and quests - Still in experimental phase.
Effects - Effects are used to apply instant or overtime modifications to entity stats.
Engine - contains the game engine
Items - Things that exist in the world and can be picked up by player
Map - tools to create maps
objects - random things that exists in the world but cant be picked up
Pickups - Items in their picked up state
StatData - Entities and weapons have base values. these are stored in statDatas and can have modifications.
